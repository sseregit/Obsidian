### 회귀 평가
- 회귀에서 오차를 계산하기 위한 평가 방법

|                   평가지표                   | 설명                                       |
| :--------------------------------------: | ---------------------------------------- |
|       MAE<br>(Mean Absolute Error)       | 실제값과 예측값 차이를 **절대값**으로 변환해 평균한 것         |
|       MSE<br>(Mean Squared Error)        | 실제값과 예측값 차이를 **제곱**해 평균한 것               |
|    RMSE<br>(Root Mean Squared Error)     | MSE에 **루트**를 씌운 것                        |
|                 R2 Score                 | 회귀 모델의 '설명력' <br>(R2 Score만 높은값이 좋다는 의미) |
|  RMSLE<br>(Root Mean Squared Log Error)  | 예측값이 실제값보다 작을때 더 큰 패널티 부여                |
| MAPE<br>(Mean Absolute Percentage Error) | MAE를 퍼센트로 표시                             |
