# 프로세스와 스레드 소개
## 멀티태스킹과 멀티프로세싱
### 시분할(Time Sharing, 시간 공유) 기법
- 각 프로그램의 실행 시간을 분할해서 마치 동시에 실행되는 것 처럼하는 기법
### 멀티태스킹(Multitasking)
- 하나의 컴퓨터 시스템이 동시에 여러 작업을 수행하는 능력
### 스케줄링(Scheduling)
- CPU에 어떤 프로그램이 얼마만큼 실행될지는 운영체제가 결정한다.
