# 효율성 증대 명령어
`--permission-mode`

| acceptEdits       | 반복적인 리팩토링, 코드정리                |
| ----------------- | ------------------------------ |
| plan              | 복잡한 기능 개발, 아키텍처 변경, 대규모 마이그레이션 |
| bypasspermissions | 도구실행보다 다양한 권한이 반복적으로 요구        |
### 주의사항
- 신뢰할수 있는 작업만
- 중요로직 변경은 수동으로 확인
- 백업 필수
- 단계적 적용
