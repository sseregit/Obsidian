# Scouter 팁
## 필수 그래픽 목록
### Tomcat/Java
- GC Count

- TPS
- Active Service EQ
- Active Speed

# 명령어
## 유닉스 기반 명령어
`ps -ef`
- 모든 pid와 프로세스 이름이 터미널에 출력된다.
`ps -ef | grep java`
- 만약 자바 프로세스를 확인하려면 다음과 같이 `| grep java`를 사용하면된다.
