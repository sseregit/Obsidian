# 플랫 IP 네트워크(flat IP network)
- 계층이나 서브네트워크 없이 모든 장치가 동일한 IP 네트워크 범위(같은 서브넷)안에 존재하는 네트워크 구조를 말한다.
- 단일 IP 서브넷 안에 있어 라우터 없이도 서로 직접 통신할 수 있는 네트워크 구조

### 장점
- 설정이 간단하고 초기 구축이 빠르다.
- 라우팅 복잡도 없음
- 저비용 네트워크 구성에 적합
### 단점
- 확장성 부족
	- 기기 수가 많아질 수록 충돌 및 브로드캐스트 폭증가능
- 보안 및 관리 기능이 부족
	- 모든 장치가 동일 평면에 있어 분리 어려움
- 장애 전파 위험 큼
	- 하나의 브로드캐스트 도메인
