# 1-2. 키 기반 접속 (다양한 키 포맷 처리)
## 1-2-1 키 기반 생성하기.
- SSH 키 인증 방식 알고리즘 RSA, ED25519 사용하기
	- 불가능 대상 서버에 OpenSSH 버전을 체크할 방법이 없음
	- RSA는 무조건 가능하니 RSA로만 테스트하기.

| 특성      | RSA (4096비트) | ED25519   |
| ------- | ------------ | --------- |
| 보안 수준   | 높음           | 매우 높음     |
| 성능      | 느림           | 빠름        |
| 키 크기    | 큼            | 작음        |
| 호환성     | 매우 높음        | 제한적일 수 있음 |
| 랜덤성 의존성 | 있음           | 없음        |
| 사용 역사   | 오래됨          | 비교적 최근    |
