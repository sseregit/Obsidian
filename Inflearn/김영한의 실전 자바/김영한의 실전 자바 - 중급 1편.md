# Object 클래스

## java.lang 패키지 소개

### java.lang
- 자바 언어를 이루는 가장 기본이 되는 클래스들을 보관하는 패키지
#### 대표적인 클래스들
##### Object
- 모든 자바 객체의 부모 클래스
##### String
- 문자열
##### Integer, Long, Double
- 래퍼 타입, 기본형 데이터 타입을 객체로 만든 것
##### Class
- 클래스 메타 정보
##### System
- 시스템과 관련된 기본 기능들을 제공

## Object 클래스

### 묵시적(Implicit) vs 명시적(Explicit)
#### 묵시적
- 개발자가 코드에 직접 기술하지 않아도 시스템 또는 컴파일러에 의해 자동으로 수행되는 것을 의미
#### 명시적
- 개발자가 코드에 직접 기술해서 작동하는 것을 의미
