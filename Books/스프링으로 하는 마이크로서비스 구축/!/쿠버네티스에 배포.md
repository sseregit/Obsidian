# 쿠버네티스에 배포
- 상용화 환경을 시뮬레이션
- 자원 관리자(MySQL, MongoDB, RabbitMQ)는 쿠버네티스 클러스터 외부에서 도커 컴포즈를 사용해 실행한다.

```bash
eval $(minikube docker-env)
docker compose up -d mongodb mysql rabbitmq
```
