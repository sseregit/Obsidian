# glob 패턴
- 파일 이름이나 경로를 지정할 때 사용하는 "간단한 와일드카드 문자 패턴"

### 대표적인 패턴 문법

| 패턴     | 의미                             |
| ------ | ------------------------------ |
| *      | 모든 문자 0개 이상                    |
| ?      | 임의의 문자 1개                      |
| [abc]  | a, b, c중 하나                    |
| [!abc] | a, b, c가 아닌 문자                 |
| **     | (특수한 경우) 하위 디렉토리 까지 포함한 재귀적 매칭 |
