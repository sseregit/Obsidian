# ComplySight 데이터베이스 엔티티 관계도 (ERD)  
  
이 문서는 `complysight-common` 모듈에 정의된 JPA 엔티티를 기반으로 ComplySight 프로젝트의 데이터베이스 스키마를 시각적으로 표현합니다.  
  
## 통합 엔티티 관계도 (Overall ERD)  
  

## 도메인별 세부 관계도  
  
### 1. 자산 및 정보시스템 관리 (Asset & System Management)  
자산이 어떻게 분류되고 정보시스템 아래에 그룹화되는지에 중점을 둡니다.  
  
```mermaid  
erDiagram  
    SYS_M ||--o{ AST_M : "포함"  
    SYS_M ||--o{ SYS_PIC_L : "관리 담당"  
    AST_M }o--|| AST_CLSF_C : "자원 유형"  
    AST_M ||--o{ AST_PIC_L : "점검 담당"  
    AST_M ||--o{ AST_INSTL_L : "설치 경로 정보"  
    AST_M ||--o{ AST_LOG_L : "로그 위치 정보"  
    AST_M ||--o{ AST_SRVC_URL_L : "서비스 URL 정보"  
      
    SYS_PIC_L }o--|| USER_M : "사용자 연결"  
    AST_PIC_L }o--|| USER_M : "사용자 연결"  
    AST_CLSF_C ||--o{ AST_CLSF_C : "상위 분류"  
```  
  
### 2. 정기 점검 및 스케줄링 (Regular Inspection & Scheduling)  
스케줄링 메커니즘과 주기적 점검 결과 기록에 대한 상세 내용입니다.  
  
```mermaid  
erDiagram  
    CHCK_SCHDL_L ||--o{ CHCK_SCHDL_AST_L : "점검 대상 설정"  
    CHCK_SCHDL_AST_L }o--|| AST_M : "대상 자원"  
    CHCK_SCHDL_L }o--|| SYS_M : "소속 시스템"  
      
    CHCK_H ||--o{ CHCK_DTL_H : "항목별 상세결과"  
    CHCK_H ||--|| CHCK_RPT_H : "최종 보고서"  
    CHCK_H ||--o{ CHCK_FILE_L : "증적 파일"  
    CHCK_RPT_H ||--o{ CHCK_RPT_FILE_L : "제출 보고서 파일"  
```  
  
### 3. 특별 점검 (Special Inspection)  
일회성 또는 특정 목적을 위한 점검 프로세스입니다.  
  
```mermaid  
erDiagram  
    CHCK_SPC_L ||--o{ CHCK_SPC_AST_L : "대상 목록"  
    CHCK_SPC_AST_L }o--|| AST_M : "대상 자원"  
    CHCK_SPC_L ||--|| CHCK_SPC_RSLT_L : "결과 기록"  
    CHCK_SPC_RSLT_L ||--o{ CHCK_SPC_FILE_L : "결과 첨부파일"  
    CHCK_SPC_L }o--|| USER_M : "담당자 배정"  
```  
  
### 4. 점검표 설정 (Checklist Settings)  
자산 유형별 점검 항목 및 필수 필드 구성 설정입니다.  
  
```mermaid  
erDiagram  
    CHLST_L ||--o{ MPNG_CHLST_VRFC_L : "검증 항목 구성"  
    MPNG_CHLST_VRFC_L }o--|| VRFC_CD_C : "검증 코드"  
    CHLST_L ||--o{ CHLST_ESNTL_AST_L : "필수 메타데이터"  
    CHLST_ESNTL_AST_L }o--|| AST_ESNTL_L : "속성 정의"  
```
