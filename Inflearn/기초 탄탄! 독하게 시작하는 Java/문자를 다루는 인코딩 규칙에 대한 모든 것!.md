
ASCII코드는 7bit 코드체계이다.

코드쳬계
- 숫자 -> 문자로 표현

영문 -> ASCII -> 동아시아(한자어) 표현이 안된다.

한글의 코드쳬계
- 조합형
	- ㄴ + ㅓ + ㄹ
	- ASCII코드 처럼 각각을 조합
- 완성형
	- 널

결과적으로 완성형으로 한글의 코드쳬계를 사용한다.
