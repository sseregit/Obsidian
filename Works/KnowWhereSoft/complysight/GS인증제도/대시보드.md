- 당일 제출된 보고서 수
```sql
select count(1)
from asset_inspection_result
where submit_dtm is not null
  and report_content is not null
  and reg_dtm >= CURRENT_DATE
  and reg_dtm < CURRENT_DATE + INTERVAL '1 day'
```

from asset_inspection_result
