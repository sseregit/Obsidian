# 쿠버네티스 API 객체
## 노드(Node)
- 노드는 클러스터에 속한 가상 혹은 물리 서버를 의미한다.
## 포드(Pod)
- 쿠버네티스에서 배포할 수 있는 컴포넌트의 최소 단위
- 하나 혹은 여러 개의 컨테이너를 포함한다.
- 컨테이너는 같은 IP 주소와 포트 범위를 공유한다.
- 일반적으로 포드는 하나의 컨테이너로 구성되지만, 주 컨테이너의 기능 확장을 위해 보조 컨테이너를 사용하기도 한다.
