### C/C++ 개발자에게 드리는 말씀
- Java는 void형 변수 선언은 불가하며 반환 형식으로만 사용
- null값은 참조형에만 적용 가능
	- int a = null (불가)
	- null은 값(인스턴스)가 없다는 의미
	- 클래스 형식을 반환하는 메서드는 null 반환 가능
- 연산자 함수 없음
- Java에서 클래스 형식은 모두 참조자이며 포인터라고 생각하는 것이 용이
